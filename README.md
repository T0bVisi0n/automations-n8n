# automations-n8n

Automation system repository for n8n workflows, integration logic, environment setup and process documentation.

## Purpose

This repository contains the automation layer of the business.

It documents, stores and structures n8n workflows, integration logic and setup requirements so automations remain understandable, reusable and maintainable over time.

## Scope

- exported n8n workflows
- workflow documentation and logic notes
- environment variable references
- API and integration setup
- automation architecture and change tracking

## Goal

Build a reliable automation system that reduces manual work, supports lead processing and creates leverage across the business.
